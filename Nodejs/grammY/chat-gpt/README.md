# Chat GPT Bot

A Telegram bot that interacts with the chatgpt-3 API

It's so easy to execute with the following steps:

1. Copy the file [.env](./.env-example) to a file `.env` editing the
   tokens they require (Telegram and OpenAI)

   ```sh
   cp .env-example .env
   vim .env
   ```

2. Run with:

```sh
npm install
npm run build
npm run start
```
